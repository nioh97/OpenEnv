# Smart Farming OpenEnv Environment

## Overview

This is a real-world reinforcement learning environment simulating farming decisions.

Agents must manage irrigation, fertilization (NPK), pest control, and harvesting under resource constraints.

## Tasks

* Easy: High resources
* Medium: Balanced constraints
* Hard: Resource-limited

## API Endpoints

* /reset
* /step
* /state
* /tasks
* /grader
* /baseline

## Deployment

Docker-based environment compatible with Hugging Face Spaces.

## Goal

Train and evaluate agents for agricultural decision-making.
